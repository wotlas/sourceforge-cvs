Contents
The wotlas-client package contains :

- The wotlas client game files v1.2 ( java classes, graphics, sounds, musics )
- The help files, access to online news.

The wotlas-server package contains :

- The wotlas server files v1.2 ( java classes, wizard steps )
- The help files.

The wotlas-devel package contains :

- The wotlas server files v1.2 ( java classes, wizard steps )
- The wotlas client game files v1.2 ( java classes, graphics, sounds, musics )
- The wotlas source code.
- The help files, access to online news.


Features

The main game features of wotlas v1.2 are :

- One world ( RandLand ), four Towns ( Tar Valon, Braem Wood, Blight Refuge, Shayol Ghul ), 14 maps.

- Seven character class : Aes Sedai, Warders, Children of the Light, Asha'man, Wolf Brother, Aiel, Darkfriend.

- Integrated chat with advanced features : threads, commands, voice level, smileys.

- Possibility to leave a message attached to one's player.

- Possibility to lie on your name.


Changes & Known Conflicts

All the changes between wotlas versions are listed here. The known conflicts are :

- Divx3.1alpha drivers cause wotlas to crash when you enter the game... please install the Divx4.0 (or better) software.

- If you experience any problems when you enter the game ( after your account creation ) please turn off the music in the option panel (available on the main screen ). We witnessed that in some cases Midi music files can make wotlas crash.


Requirements

Hardware requirements on the client side :

- A 300Mhz processor minimum ( 500Mhz or more strongly recommended ), 64 Mo RAM.
- An Internet connection ( 56k Modem or better ).
- A sound card with Midi support.

Hardware requirements on the server side :

- A 500Mhz processor minimum ( 700Mhz or more strongly recommended ), 128 Mo RAM.
- A fast Internet connection ( DSL or better ).
- Optional : a sound card ( only necessary if you want to use the server bell feature ).


         Wotlas has been tested successfully on Windows OS ( 98, Me, NT4, 2000 & XP ) and Linux OS. We also tested wotlas using Java 1.3.0, 1.3.1 and 1.4 . Some minor bugs that have not been fixed yet are listed in our FAQ.


Installation

1 - If you have an old version of Java ( 1.1, 1.2, 1.2.1 ) first un-install it ( for windows check the Java Plug-in in the configuration panel, for unix enter 'java -version' in a shell window ). Wotlas works only with Java 1.3 or higher.

2 - If you have to install Java on your computer here are the links :

Java Runtime Environment 1.3.1 for Windows
Java Runtime Environment 1.3.1 for Linux
Other Versions of Java

3 - For Windows users : restart your computer after Java installation.

4 - On the client side :

           Start the game with wotlas.bat ( Windows ) or wotlas.sh ( Unix ). If you encounter any problem first read our FAQ . If your problem is not listed here please contact us. If no wotlas server seems to be running please check our news section.

     On the server side :

           Start the register utility with register.bat ( Windows ) or register.sh ( Unix ). It will help you configure your server. After this step you'll have to declare you server's IP/DNS address with the setup.bat ( Windows ) or setup.sh ( Unix ) scripts. Each time your server's address changes you'll have to call the setup utility.

           You'll then be able to start your server with the server.bat ( Windows ) or server.sh ( Unix ) scripts. If you encounter any problem first read the FAQ . If your problem is not listed here please contact us.
