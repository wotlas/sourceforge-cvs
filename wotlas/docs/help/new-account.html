How do I create a new account ?

When you start the game, the first window displayed is the New Account Window. Just enter a login, enter twice a password ( min 5 characters ) and select a game server.

When you click on a game server a pop-up window appears and shows some information on the selected server.

When you click on the Ok button a connection with the selected server will established. If an error occur at this step it might either mean that :

the server is not running. try to select another server ( if there is one ).

your Internet connection might not be running anymore... ( easy to check... )

If the connection succeeds you'll have to fill different forms. Take some time to create a realistic Wheel of Time character.

When your account has been created you are given a key and can directly enter the game.
