ABOUT:
-----

This path contains:

- Chat : a chat Client/Server example implemented on top of the wotlas.net package
         Main classes are Client.java and Server.java

- Performance : client & server programs to test the performance of the wotlas.net package
                Main classes are Client.java and Server.java


HOW TO USE:
----------

- checkout the package you are interested in...

- move it in your src/ directory where the wotlas package are.
 ( you can also modify your classpath if you don't want to move these files)

- compile and run...
