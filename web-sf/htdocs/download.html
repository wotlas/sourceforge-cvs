We provide many ways to install Wotlas. The easiest of all is via Java Web Start (JWS), a new application-deployment technology that gives you the power to launch full-featured applications with a single click from your Web browser. JWS includes the security features of the Java2 platform, so the integrity of your data and files is never compromised.

          To install wotlas, you just need to click on the two images below. If you want other installation solutions, zip packages are available here.

          If nothing happens when you click on the image above please make sure JWS is installed correclty ( troubleshooting help here ). If you encounter any problem during the installation process read the release notes or read the FAQ we provide. Contact us on our forums if you have any questions.
