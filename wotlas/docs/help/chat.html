Chat

Here is how our wotlas chat works. If you find it hard to use in the game please let us know ! Here we go :

On building maps chat channels are attached to rooms, you can only discuss with players of the same room/place.

You might have noticed that there is always a default chat channel anywhere you go. This channel represents the place where you are. When you speak in it your messages are sent to everybody in the room.

You can create an other chat channel if you want to discuss in private with some other players. Just click the icon representing an empty document with a star. This user channel will be seen only by the players of your current room/location.

User chat channels have a completely different behaviour from the default chat channel. To join a user chat channel you have to be near someone who is using it, i.e. you need to move near a group of players to be able to communicate on their chat channel. Logical isn't it ?

If you move toward a player that has a user chat channel your current chat channel will change automatically to his/her.

To leave a user chat channel you can either hit the Leave button or move away from the players that are using the channel. When the last member of a user channel moves away, the channel is automatically destroyed... except if he/she is the creator of the chat channel. This way the creator of a user channel can move with his/her channel and if he/she comes near a player who is on the default chat channel, this player will have our channel as the currently selected one.

You might have noticed your player speech volume on the bottom-left of the chat. There are three volume level : whisper (on the left), normal (default), shout (on the right).

When you shout everybody in your room and the near rooms receive your message on their current chat channel.

When you enter a new user chat channel you are not considered as an active member of the user channel. It means that you'll be able to hear what's said but not what's whispered ! To become an active member of a user chat channel just send a message to it ! you will then hear everything, even whispers.

If you don't want people that move around your group to hear what you say in your user chat channel you should whisper... this way other people won't be able to eavesdrop without you knowing.

Chat Shortcuts & Commands

You can use the following shortcuts in the chat prompt (your message must begin with them):

/shout - to shout

/whisper - to whisper

/me - to begin a sentence with : '[your name] says ...'

We also provide some commands you can use in the chat :

/fanfare, /knock, /bell - produce a sound to draw attention. The fanfare sound is not the same for each character class.

/who - to display the list of connected players

/help - to display all the available commands. You can also click on the chat /? icon.

/server - to get some information on the current server.

/info - command documentation. For example to get some information on the /who command enter /info:/who

Finally, if you enter *doh* or *toh* in a chat message it will produce the famous Homer Simpson's sounds...
