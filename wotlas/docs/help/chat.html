Chat

Here is how work our wotlas chat. We know there are lots of rules, so, if you find some of them hard to use in the game please let us know ! Here we go :

On building maps chat channels are attached to rooms, you can only discuss with players of the same room/place.

You might have notice that there is always a default chat anywhere you go. This channel represents the place where you are. When you speak in this default channel your messages are sent to everybody in the room.

You can create an other chat channel if you want to discuss in private with some other players. Just hit the button representing an empty document with a *.This user channel will be seen only by the players of your current room.

User chat channels have a completely different behaviour from the default chat. To join a user chat channel you have to be near someone who is using it. That means that you need to move near a group of players to be able to communicate on their user chat channel.

If you move toward a player that has a user chat your current chat channel will be automaticaly changed to his/her.

To leave a user chat channel you can either hit the Leave button or move away from the players that are using the channel. When the last member of a user channel moves away, the channel is automatically destroyed... except if he/she's the creator of the chat channel. This way the creator of a user channel can move with his/her channel and if he/she comes near a player who is on the default chat, this player will have our channel as the currently selected one.

You might have notice your player speach volume on the bottom-left of the chat. There are three volume level : whisper (on the left), normal (default), shout (on the right).

When you shout everybody in the room and in the near rooms receive your message on their current chat.

When you enter a new user chat channel you are not considered as an active member of the user channel. It means that you'll be able to hear what's said but not what's whispered ! To become an active member of a user chat channel just send a message to it ! you will then hear everything, even whispers.

If you don't want people that moves near your group to hear what you say in your user chat channel you should whisper... this way other people won't be able to eavesdrop without you knowing.

Chat Shortcuts

You can use the following shortcuts in the chat prompt (your message must begin with them):

/shout - to shout

/whisper - to whisper

/me - to begin a sentence with : '[your name] says ...'
